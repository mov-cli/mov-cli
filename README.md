# mov-cli

A cli to browse and watch movies.

## Installation
This project is a work in progress.
However, you can try it out

### python

```bash
git clone https://github.com/mov-cli/mov-cli
cd mov-cli
python setup.py install
```
### Shell

```bash
git clone https://github.com/mov-cli/mov-cli
cd mov-cli
./mov-cli
```
